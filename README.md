## 📝 Описание программы

Bank Account System - это консольное приложение для управления банковскими счетами, разработанное с использованием принципов ООП и паттерна MVC. Программа поддерживает различные типы счетов (текущий, сберегательный, кредитный) и позволяет выполнять основные банковские операции.

### Основные возможности:

- ✅ Создание различных типов счетов (Checking, Savings, Credit)
- ✅ Пополнение и снятие средств
- ✅ Перевод средств между счетами
- ✅ История транзакций с использованием очереди
- ✅ Фильтрация транзакций по дате и типу
- ✅ Начисление процентов на сберегательный счет
- ✅ Сохранение и загрузка данных в JSON
- ✅ Обработка ошибок (недостаточно средств, некорректный ввод)
